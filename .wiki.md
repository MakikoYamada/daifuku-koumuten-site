# Project Summary
This project is a web application designed to facilitate user inquiries and contact submissions, streamlining communication between users and the service provider. It features a well-structured frontend interface built with React and TypeScript, allowing efficient handling of queries. A minimal backend has been reintroduced to support essential functionalities, including a health check endpoint.

# Project Module Description
The project consists of two modules: the frontend and the backend.

- **Frontend**: 
  - Provides a user interface for users to submit inquiries and view information.
  - Implements various components for form handling and displaying results.

- **Backend**:
  - Minimal FastAPI application to handle basic requests.
  - Includes a `/health` endpoint for health checks.

# File Description Inventory
- **frontend/src/components/ContactSection.tsx**: React component for the contact form interface, now simplified to a client-side form with an alert on submission.
- **backend/main.py**: Minimal FastAPI app with a `/health` endpoint and CORS middleware.
- **backend/requirements.txt**: Lists backend dependencies (FastAPI, Uvicorn).
- **backend/requirements.default**: Default backend dependencies (FastAPI, Uvicorn).

# Technology Stack
- **Frontend**: React, TypeScript, Vite
- **Backend**: FastAPI, Uvicorn
- **Deployment**: AWS (for serverless functions)

# Usage
To set up the project, follow these steps:

1. **Install Dependencies**:
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   - For the backend: Navigate to `app/backend` and install dependencies listed in `requirements.txt`.

2. **Build the Project**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run the Project**:
   - Start the frontend development server by running `pnpm run dev` from the `app/frontend` directory.
   - Start the backend server using Uvicorn to serve the FastAPI app.
